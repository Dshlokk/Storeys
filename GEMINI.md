# Project: Storeys Realty

## Tech Stack
- **Frontend:** React (Vite)
- **Styling:** CSS Modules or plain CSS (as seen in `src/components`)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Routing:** React Router DOM
- **Backend:** Node.js (Express)

## Architecture
- `src/components`: Reusable UI components.
- `src/pages`: Page-level components associated with routes.
- `public`: Static assets.

## Conventions
- **Components:** Functional components with React hooks.
- **Naming:** PascalCase for component files and folders.
- **Styles:** Each component has a corresponding `.css` file in the same directory.
- **Imports:** Prefer absolute paths or relative paths as per Vite config (to be verified).

## Development Workflow
- `npm run dev`: Start the Vite development server.
- `npm run build`: Build for production.
